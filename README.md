# DripKart 👟🔥

DripKart is a modern fashion and sneaker e-commerce platform built for streetwear enthusiasts.  
The platform allows users to explore trending footwear, clothing, and accessories with a smooth and responsive shopping experience.

## 🚀 Features

- User Authentication
- Product Listings
- Search & Filters
- Shopping Cart
- Wishlist
- Secure Checkout
- Responsive UI
- Admin Dashboard
- Order Management

## 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## 📁 Project Structure

- `frontend` → React + Vite storefront UI
- `backend` → API/server folder (backend implementation target)

## ▶️ Run Frontend

```bash
npm --prefix frontend install
npm --prefix frontend run dev
```
